## 📊 监控指标

### 基本面指标
- 最新价、涨跌幅

### 技术指标

## ⚙️ 配置说明

### 默认配置
- **止盈线**: 15%
- **止损线**: -8%
- **最大持仓天数**: 30天

### 修改配置

编辑配置文件：
```bash
~/.stock_monitor/002837/config.json
```

或修改代码中的默认值：
```python
self.config = {
    "take_profit": 15.0,  # 止盈%
    "stop_loss": -8.0,  # 止损%
    "max_hold_days": 30,  # 最大持仓天数
}
```

## 📱 Telegram提醒

系统会自动发送以下提醒到你的Telegram：

### 止盈提醒
当盈亏达到止盈线时发送：
```
🔴 止盈提醒！当前盈亏XX%，已达到止盈线XX%，建议考虑止盈锁定利润。
```

### 止损提醒
当盈亏达到止损线时发送：
```
🔴 止损提醒！当前盈亏XX%，已达到止损线XX%，建议止损控制风险。
```

### 持仓时间提醒
当持仓时间接近最大持仓期时发送：
```
🟡 持仓时间提醒！已持仓XX天，接近最大持仓期XX天，建议评估是否继续持有。
```

## 🔄 定时监控


添加以下行（每30分钟检查一次）：
```bash

```

或者每个交易日收盘后检查：
```bash
0 15 * * 1-5 /Users/wtueeq/.openclaw/workspace/check_alerts_002837.py >> /Users/wtueeq/.stock_monitor/002837/cron.log 2>&1
```

## 🔧 故障排除

### 问题1：无法获取实时数据
**原因**: 网络连接问题或AkShare数据源问题
**解决**: 检查网络连接，稍后重试

### 问题2：Telegram提醒收不到
**原因**: Bot Token或Chat ID配置错误
**解决**: 检查Telegram配置

### 问题3：技术指标计算错误
**原因**: 历史数据不足
**解决**: 确保有足够的历史数据（至少20天）
