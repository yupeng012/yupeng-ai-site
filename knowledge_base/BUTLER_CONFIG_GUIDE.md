# 📱 管家配置说明

## 🎯 功能说明

管家功能会将所有股票和金融相关的内容自动发送到指定的Telegram频道或群组，方便你集中管理。

## ⚙️ 配置步骤

### 第一步：获取管家Chat ID

1. **如果是Telegram频道**
   - 创建一个Telegram频道
   - 将你的bot添加为频道管理员
   - 频道的Chat ID通常是负数，如：`-1001234567890`

2. **如果是Telegram群组**
   - 创建一个Telegram群组
   - 将你的bot添加到群组
   - 群组的Chat ID通常是负数，如：`-1234567890`

3. **如何获取Chat ID**
   - 方法1：使用@userinfobot机器人
   - 方法2：发送消息到频道/群组，然后访问：
     ```
     https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
     ```
   - 方法3：使用第三方工具，如：https://t.me/MyTelegramID_bot

### 第二步：配置管家

编辑 `butler_config.py` 文件：

```python
BUTLER_CONFIG = {
    "enabled": True,
    "name": "管家",
    "telegram_chat_id": "-1001234567890",  # 替换为你的Chat ID
    "telegram_bot_token": "8205207757:AAHBTHZXeRjDLZwzaO4uZzZZsNW6lqqwkdQ",
    ...
}
```

### 第三步：测试配置

运行测试脚本：

```bash
python3 butler_config.py
```

如果配置正确，你会收到测试消息。

## 📊 发送内容类型

管家会自动发送以下内容：

### 1. 每日报告 (daily_report)
- 每天早上9:00自动发送
- 包含股票分析报告
- 持仓盈亏情况
- 技术指标分析

### 2. 操作提醒 (alert)
- 止盈提醒
- 止损提醒
- 持仓时间提醒
- 技术信号提醒

### 3. 分析报告 (analysis)
- 完整的技术分析
- 基本面分析
- 操作建议

### 4. 市场更新 (market_update)
- 每天下午3:00发送
- 市场整体情况
- 板块动态

### 5. 持仓更新 (position_update)
- 持仓变化
- 盈亏更新
- 操作记录

## 🔄 发送时间配置

可以在 `butler_config.py` 中修改发送时间：

```python
"schedule": {
    "daily_report": "09:00",  # 每日报告发送时间
    "market_update": "15:00"  # 市场更新发送时间
}
```

## 🎨 自定义配置

### 修改发送内容类型

编辑 `butler_config.py`：

```python
"send_types": [
    "daily_report",      # 每日报告
    "alert",             # 操作提醒
    "analysis",          # 分析报告
    # "market_update",   # 市场更新（注释掉则不发送）
    # "position_update"  # 持仓更新（注释掉则不发送）
]
```

### 启用/禁用管家功能

```python
BUTLER_CONFIG = {
    "enabled": False,  # 设置为False禁用
    ...
}
```

## 🔧 故障排除

### 问题1：收不到消息

**可能原因：**
1. Chat ID配置错误
2. Bot没有管理员权限
3. 网络连接问题

**解决方法：**
1. 检查Chat ID是否正确
2. 确保Bot是频道/群组管理员
3. 运行测试脚本验证配置

### 问题2：消息格式错误

**可能原因：**
1. 消息内容包含特殊字符
2. 消息过长

**解决方法：**
1. 检查消息内容
2. 分段发送长消息

### 问题3：发送频率过高

**可能原因：**
1. 定时任务配置错误
2. 重复发送

**解决方法：**
1. 检查crontab配置
2. 添加发送记录，避免重复
