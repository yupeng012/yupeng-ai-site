# 多智能体单机器人配置方案

## 智能体列表

1. **main** - 🐟 小鱼儿
2. **stock-analyst** - 📊 股票分析师
3. **待创建智能体** - 需要用户指定

## 配置方案

### 1. 消息前缀配置
每个智能体在发送消息时，自动添加身份标识：

- main: `🐟 小鱼儿：`
- stock-analyst: `📊 股票分析师：`

### 2. Telegram机器人配置
使用现有的机器人（yuzejing_bot），配置为：
- 接收所有智能体的消息
- 根据消息内容路由到对应的智能体
- 在回复中标注智能体身份

### 3. 实现方式

#### 方式1：使用Agent Routing（推荐）
在openclaw.json中配置agent routing规则，让不同的智能体能够通过同一个Telegram机器人发送消息。

#### 方式2：使用消息前缀
在每个智能体的system prompt中添加消息前缀配置，让智能体在发送消息时自动添加身份标识。
