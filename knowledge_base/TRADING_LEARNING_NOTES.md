## 📈 基本面分析学习

### 财务指标

#### ROE（净资产收益率）
- [ ] ROE > 15%：优秀
- [ ] ROE 10-15%：良好
- [ ] ROE 5-10%：一般
- [ ] ROE < 5%：较差

#### 毛利率
- [ ] 毛利率 > 40%：高毛利
- [ ] 毛利率 30-40%：中高毛利
- [ ] 毛利率 20-30%：中等毛利
- [ ] 毛利率 < 20%：低毛利

#### 净利率
- [ ] 净利率 > 15%：高净利
- [ ] 净利率 10-15%：良好净利
- [ ] 净利率 5-10%：一般净利
- [ ] 净利率 < 5%：低净利

#### 现金流
- [ ] 经营现金流 > 0：现金流为正
- [ ] 经营现金流 < 0：现金流为负
- [ ] 现金流与利润匹配：质量好
- [ ] 现金流与利润不匹配：质量差

### 估值方法

#### PE估值（市盈率）
- [ ] PE < 20：低估值
- [ ] PE 20-40：合理估值
- [ ] PE 40-60：偏高估值
- [ ] PE > 60：高估值

#### PB估值（市净率）
- [ ] PB < 1.5：低估值
- [ ] PB 1.5-3：合理估值
- [ ] PB 3-5：偏高估值
- [ ] PB > 5：高估值

#### PS估值（市销率）
- [ ] PS < 5：低估值
- [ ] PS 5-10：合理估值
- [ ] PS 10-20：偏高估值
- [ ] PS > 20：高估值

#### PEG估值（市盈率相对盈利增长）
- [ ] PEG < 1：低估
- [ ] PEG 1-2：合理
- [ ] PEG 2-3：高估
- [ ] PEG > 3：高估

### 行业对比

#### 行业增速
- [ ] 行业增速 > 20%：高增长
- [ ] 行业增速 10-20%：中增长
- [ ] 行业增速 0-10%：低增长
- [ ] 行业增速 < 0：负增长

#### 竞争格局
- [ ] 垄断性行业：高壁垒
- [ ] 竞争性行业：中壁垒
- [] 竞争激烈行业：低壁垒
- [ ] 完全竞争行业：无壁垒

#### 公司定位
- [ ] 行业龙头：市场份额第一
- [ ] 行业领先者：市场份额前三
- [ ] 行业跟随者：市场份额前十
- [ ] 行业挑战者：市场份额前十外
